Quick Manual

Open help from the Artic crystal button at the top right. Use this tab as a quick guide for loading data, moving around, editing annotations, and exporting results.

Open Data

Action What to do
Drag and drop Drop audio or video files, optionally paired with .TextGrid or annotJSON files, into the start screen or click the drop area to choose local files.
open demo Use open demo in the toolbar to load one of the built-in example databases.
connect Use connect to attach to an EMU WebSocket server and browse its bundles.

Navigate

Area What it is for
Bundle list Open the left sidebar to switch between sessions and bundles in the loaded database.
Zoom and pan Zoom into detail, zoom back out, and pan through time in the signal view to focus on the region you need.
Perspectives Open the perspectives sidebar to switch between different signal and annotation layouts for the same data.
Hierarchy Open hierarchy to inspect and adjust links between related annotation levels and items.

Edit And Listen

Task How it works
Select Click items or move the highlighted selection to choose segments, events, or levels before editing.
Edit Add boundaries or events, relabel items, adjust timing, and undo or redo changes as needed.
Listen Play the visible region, a selected region, or the whole file to check timing and labels while you work.
Shortcuts This tab stays high level; open Keys for the full keyboard reference.

Save And Export

Option Result
save If save is available, it writes the current bundle back to its source after editing.
TextGrid Export the current bundle as a Praat TextGrid for use outside Artic.
annotJSON Export the current bundle as annotation JSON, including the annotation structure used by Artic.
